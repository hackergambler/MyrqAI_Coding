# Project: AI Atlas & Coding Academy (coding.myrqai.com)

## 🎯 Global Mission
Build a world-class, 4-tier AI training resource. Every tier must have at least 50 detailed sub-topics. No empty links allowed.

## 📂 Comprehensive 4-Tier Roadmap (Build These Categories)

### Level 1: Foundations (The "Ground Zero")
- **Python for AI:** Variables, Data Structures (Lists, Dicts, Tensors), List Comprehensions.
- **Math for ML:** Linear Algebra (Matrices), Calculus (Gradients), Statistics (Probability).
- **Data Prep:** NumPy, Pandas, Data Cleaning, Normalization, Data Augmentation.

### Level 2: Machine Learning (The "Core")
- **Supervised:** Regression, Random Forests, XGBoost, SVM.
- **Unsupervised:** K-Means, PCA, Anomaly Detection.
- **Evaluation:** Bias-Variance Tradeoff, F1-Score, ROC Curves, Cross-Validation.

### Level 3: Deep Learning (The "Engine")
- **Architectures:** CNNs (Vision), RNNs/LSTMs (Time-series), GANs (Image Gen).
- **Frameworks:** PyTorch (Tensors, Autograd), TensorFlow (Keras API).
- **Optimization:** Stochastic Gradient Descent, Adam, Dropout, Batch Norm.

### Level 4: Modern AI (The "Expert")
- **Transformers:** Attention Mechanisms, BERT, GPT Architecture.
- **LLM Engineering:** Prompt Engineering, Fine-tuning (LoRA/QLoRA), RAG (Vector DBs).
- **Agentic AI:** LangChain, AutoGPT, CrewAI, Function Calling, Tool Use.
- **MLOps:** Docker, Kubernetes for AI, Model Deployment, Monitoring.

## 🏗 Technical Requirements
- **Scaling:** Use `ContentEngine.js` to pull from a massive `master_library.json`. 
- **Interactive UI:** Implement the "Debugging Lab" with split-pane code editors.
- **Navigation:** The Sidebar must be a nested accordion reflecting the 4 levels above.
- **SEO & Ads:** Ensure high-commercial intent keywords are in headers for AdSense optimization.

## 🤖 Workflow Rules
- **Autonomy:** Use `--yes` and `--dangerously-skip-permissions` to work without stopping.
- **QA:** Check every `href` before committing. If a link points to a non-existent file, CREATE that file.
- **Persistence:** If credits run low or rate limits hit, sleep and resume until `finished.txt` is created.
