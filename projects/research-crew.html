Project: Research Crew Agent

Build a multi-agent research team with parallel execution: Searcher, Synthesizer, Fact-Checker, and Writer.

What You Will Build

A multi-agent research team: Searcher finds facts, Synthesizer extracts insights, Fact-Checker validates claims, Writer produces the final report. Uses asyncio for parallel execution.

Python - research crew with parallel agents
from openai import OpenAI, AsyncOpenAI
import asyncio, json

client = OpenAI()

def run_agent(name: str, system: str, task: str, model: str = "gpt-4o-mini") -> str:
    resp = client.chat.completions.create(
        model=model,
        messages=[{"role":"system","content":system},{"role":"user","content":task}]
    )
    result = resp.choices[0].message.content
    print(f"[{name}] Done ({resp.usage.total_tokens} tokens)")
    return result

AGENTS = {
    "searcher":     "You are a researcher. Find key facts and statistics. Return bullet points.",
    "synthesizer":  "You are an analyst. Extract the 5 most important non-obvious insights.",
    "fact_checker": "You are a fact-checker. Flag claims that need verification.",
    "writer":       "You are a technical writer. Write a clear, structured report.",
}

async def research_crew(topic: str) -> str:
    print(f"\nResearching: {topic}\n")

    # Step 1: Research (sequential, needs quality)
    research = run_agent("Searcher", AGENTS["searcher"],
        f"Research thoroughly: {topic}", model="gpt-4o")

    # Steps 2+3: Run in parallel
    async_client = AsyncOpenAI()

    async def async_agent(name, system, task):
        resp = await async_client.chat.completions.create(
            model="gpt-4o-mini",
            messages=[{"role":"system","content":system},{"role":"user","content":task}]
        )
        print(f"[{name}] Done")
        return resp.choices[0].message.content

    synthesis, fact_check = await asyncio.gather(
        async_agent("Synthesizer", AGENTS["synthesizer"], f"Insights from:\n{research}"),
        async_agent("Fact-Checker", AGENTS["fact_checker"], f"Check:\n{research}"),
    )

    # Step 4: Final report
    report = run_agent("Writer", AGENTS["writer"],
        f"Topic: {topic}\nResearch:\n{research}\nInsights:\n{synthesis}\nFlags:\n{fact_check}",
        model="gpt-4o")
    return report

report = asyncio.run(research_crew("AI agents in software development 2025"))
print(report)
💡 Cost optimization

Use gpt-4o for the Searcher and Writer (quality matters), but gpt-4o-mini for Synthesizer and Fact-Checker (speed + cost). This cuts cost by ~70% vs all-gpt-4o.

Step-by-Step Guide

  1. Install dependencies — pip install openai and set OPENAI_API_KEY in your environment.
  2. Copy the code above into research_crew.py.
  3. Run it — python research_crew.py. The Searcher runs first (gpt-4o, ~15s), then the Synthesizer and Fact-Checker run in parallel (~8s together), then the Writer produces the final report.
  4. Swap the topic — change the last line to any research question: research_crew("LLM fine-tuning best practices 2026").
  5. Save to file — add Path("report.md").write_text(report) after the final line to persist results.

How It Works

The crew uses a fan-out / fan-in pattern. The Searcher runs first because the Synthesizer and Fact-Checker both need its output. Then asyncio.gather() runs those two agents simultaneously — cutting total time roughly in half compared to sequential execution. Finally the Writer receives all three outputs and composes the report.

Text - execution timeline
0s   Searcher starts    (gpt-4o, ~15s)
15s  Synthesizer starts │ Fact-Checker starts  (parallel, ~8s each)
23s  Writer starts with all outputs           (gpt-4o, ~12s)
35s  Final report ready
