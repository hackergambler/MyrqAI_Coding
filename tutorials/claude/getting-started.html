Getting Started with Claude API

Install Anthropic SDK, make your first Claude call, use system prompts, manage multi-turn conversations, and stream responses.

1. Install and Configure

bash
pip install anthropic
export ANTHROPIC_API_KEY="sk-ant-..."

2. Your First Claude Message

Python - minimal Claude API call
import anthropic

# Reads ANTHROPIC_API_KEY from environment automatically
client = anthropic.Anthropic()

message = client.messages.create(
    model="claude-opus-4-5",
    max_tokens=1024,
    messages=[{"role": "user", "content": "Explain recursion in 3 sentences."}]
)

print(message.content[0].text)
print(f"Model: {message.model}")
print(f"Input tokens: {message.usage.input_tokens}")
print(f"Output tokens: {message.usage.output_tokens}")
print(f"Stop reason: {message.stop_reason}")

3. System Prompts

Claude uses a top-level system parameter - not a system message in the messages array.

Python - system prompt
message = client.messages.create(
    model="claude-opus-4-5",
    max_tokens=512,
    system="You are a concise Python tutor. Give short code examples. No preambles.",
    messages=[{"role": "user", "content": "How do I open a file?"}]
)
print(message.content[0].text)

4. Multi-turn Conversations

Python - conversation history
messages = [
    {"role": "user", "content": "What is a Python generator?"},
    {"role": "assistant", "content": "A generator is a function that uses yield to produce values lazily."},
    {"role": "user", "content": "Show me a real example."}  # remembers prior turns
]

resp = client.messages.create(
    model="claude-opus-4-5", max_tokens=512, messages=messages
)
print(resp.content[0].text)

5. Streaming

Python - streaming response tokens
with client.messages.stream(
    model="claude-opus-4-5",
    max_tokens=512,
    messages=[{"role": "user", "content": "Write a haiku about programming."}]
) as stream:
    for text in stream.text_stream:
        print(text, end="", flush=True)

final = stream.get_final_message()
print(f"\nTotal tokens: {final.usage.input_tokens + final.usage.output_tokens}")

6. Model Selection Guide

  • claude-opus-4-5 - Most capable. Complex reasoning, long-form writing, analysis. $15/$75 per MTok.
  • claude-sonnet-4-6 - Balanced. Best for most production tasks. $3/$15 per MTok.
  • claude-haiku-4-5-20251001 - Fastest, cheapest. High-volume simple tasks. $0.25/$1.25 per MTok.
Python - choose model based on task
def get_model(complexity: str) -> str:
    return {
        "simple":  "claude-haiku-4-5-20251001",
        "normal":  "claude-sonnet-4-6",
        "complex": "claude-opus-4-5",
    }.get(complexity, "claude-sonnet-4-6")
💡 Cost tip

Use Haiku for all prototyping and testing - it is 60x cheaper than Opus. Switch to Sonnet or Opus only when quality requires it.
