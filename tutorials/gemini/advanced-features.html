Gemini Advanced Features — JSON Mode, Grounding & Caching

Use Gemini's JSON output mode, search grounding, context caching, and code execution for production apps.

JSON Output Mode

Force Gemini to always return valid JSON by setting the response MIME type.

Python - guaranteed JSON output
import google.generativeai as genai
import json

model = genai.GenerativeModel(
    "gemini-1.5-flash",
    generation_config=genai.GenerationConfig(
        response_mime_type="application/json"
    )
)

response = model.generate_content(
    """Extract: name, email, company from this text.
    Return as JSON with those exact keys.
    Text: Hi I'm Sarah Lee, sarah@techco.com, TechCo Inc"""
)
data = json.loads(response.text)
print(data)  # {"name": "Sarah Lee", "email": "sarah@techco.com", ...}

Context Caching — Save Costs on Long Docs

Cache large context (system prompts, documents) so you only pay for it once across many requests.

Python - context caching for repeated queries
import datetime

# Cache a large document — pay for upload once
cache = genai.caching.CachedContent.create(
    model="models/gemini-1.5-flash-001",
    system_instruction="You are an expert on this document.",
    contents=[{
        "role": "user",
        "parts": [{"text": open("large_document.txt").read()}]
    }],
    ttl=datetime.timedelta(hours=1)  # cache for 1 hour
)
print(f"Cached: {cache.name}")

# Use cached content for multiple queries — much cheaper
model = genai.GenerativeModel.from_cached_content(cached_content=cache)
for question in ["What is the main topic?", "Summarize section 2", "What are the conclusions?"]:
    resp = model.generate_content(question)
    print(f"Q: {question}\nA: {resp.text[:100]}\n")

Google Search Grounding

Let Gemini search the web for current information and cite sources.

Python - search grounding for current facts
from google.generativeai.types import Tool, GoogleSearchRetrieval

model = genai.GenerativeModel(
    "gemini-1.5-flash",
    tools=[Tool(google_search_retrieval=GoogleSearchRetrieval())]
)

response = model.generate_content(
    "What are the latest Python 3.13 features released in 2024?"
)
print(response.text)

# Check grounding metadata (sources used)
if response.candidates[0].grounding_metadata:
    sources = response.candidates[0].grounding_metadata.web_search_queries
    print(f"Searched for: {sources}")
💡 Caching savings

Cached tokens cost 75% less than normal input tokens. If you have a 100K-token system prompt queried 100 times per day, caching saves ~$100/day on Gemini 1.5 Pro.
