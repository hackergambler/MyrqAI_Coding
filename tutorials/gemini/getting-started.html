Getting Started with Gemini API

Set up Google AI Studio, obtain your API key, install the SDK, and make your first Gemini call in both Python and JavaScript.

1. Get Your API Key

Go to Google AI Studio at aistudio.google.com, sign in with your Google account, and click Get API Key → Create API key. Copy it — you'll use it in every request.

⚠️ Free Tier Limits

Gemini 1.5 Flash is free up to 15 RPM and 1M tokens/day. gemini-1.5-pro has lower free limits. Check ai.google.dev/pricing for current rates.

2. Install the SDK

bash
# Python
pip install google-generativeai

# Node.js
npm install @google/generative-ai

# Store your key safely
export GOOGLE_API_KEY="AIza..."   # bash
# Or use a .env file + python-dotenv

3. Your First API Call

Python - minimal Gemini call
import google.generativeai as genai
import os

genai.configure(api_key=os.environ["GOOGLE_API_KEY"])

model = genai.GenerativeModel("gemini-1.5-flash")
response = model.generate_content("Explain how neural networks learn in 3 sentences.")

print(response.text)
print(f"Finish reason: {response.candidates[0].finish_reason}")
print(f"Token count: {response.usage_metadata.total_token_count}")
JavaScript — Gemini with Node.js
const { GoogleGenerativeAI } = require("@google/generative-ai");

const genAI = new GoogleGenerativeAI(process.env.GOOGLE_API_KEY);

async function main() {
  const model = genAI.getGenerativeModel({ model: "gemini-1.5-flash" });
  const result = await model.generateContent("Write a haiku about async programming.");
  console.log(result.response.text());
}
main();

4. Model Selection

  • gemini-1.5-flash — Fast, cheap, 1M token context. Best for most tasks.
  • gemini-1.5-pro — Highest quality, 2M token context. Use for complex reasoning.
  • gemini-1.5-flash-8b — Smallest and fastest. Great for classification/simple tasks.
  • gemini-2.0-flash — Latest generation, fast and capable multimodal model.
Python — compare models side by side
models_to_try = [
    "gemini-1.5-flash",
    "gemini-1.5-pro",
    "gemini-2.0-flash",
]
prompt = "Write a one-line Python function to check if a number is prime."

for model_name in models_to_try:
    model = genai.GenerativeModel(model_name)
    resp = model.generate_content(prompt)
    tokens = resp.usage_metadata.total_token_count
    print(f"\n[{model_name}] ({tokens} tokens)")
    print(resp.text[:200])

5. Generation Config

Python — control temperature, tokens, and stops
generation_config = genai.GenerationConfig(
    temperature=0.7,        # 0=deterministic, 1=creative, 2=max
    top_p=0.95,
    top_k=64,
    max_output_tokens=512,
    stop_sequences=["###", "END"],
    response_mime_type="text/plain",   # or "application/json"
)

model = genai.GenerativeModel(
    model_name="gemini-1.5-flash",
    generation_config=generation_config,
    system_instruction="You are a senior Python engineer. Always include type hints."
)

response = model.generate_content("Write a function to merge two sorted lists.")
print(response.text)
💡 System Instructions

Gemini uses system_instruction at model creation time (not per-call like OpenAI). Create one model instance per persona and reuse it.
