Gemini Multimodal — Images, Audio & Video

Gemini is natively multimodal. Send images, audio clips, and video to analyze visuals, transcribe speech, summarize videos, and extract structured data.

Sending Images

Python — analyze image from URL
import google.generativeai as genai
import httpx, PIL.Image
from io import BytesIO

model = genai.GenerativeModel("gemini-1.5-flash")

# From URL: download and pass PIL image
img_bytes = httpx.get("https://upload.wikimedia.org/wikipedia/commons/thumb/4/47/PNG_transparency_demonstration_1.png/280px-PNG_transparency_demonstration_1.png").content
img = PIL.Image.open(BytesIO(img_bytes))

response = model.generate_content([
    img,
    "Describe this image in detail. What do you see?"
])
print(response.text)
Python — analyze local image file
import PIL.Image

# From local file
img = PIL.Image.open("screenshot.png")

response = model.generate_content([
    "Extract all text visible in this screenshot.",
    img
])
print(response.text)

# Multiple images in one prompt
img1 = PIL.Image.open("before.jpg")
img2 = PIL.Image.open("after.jpg")

response = model.generate_content([
    "What changed between image 1 and image 2?",
    "Image 1:", img1,
    "Image 2:", img2
])
print(response.text)
Python — extract structured data from image
import json

# Force JSON output from an image
model_json = genai.GenerativeModel(
    "gemini-1.5-flash",
    generation_config=genai.GenerationConfig(response_mime_type="application/json")
)

img = PIL.Image.open("receipt.jpg")
response = model_json.generate_content([
    """Extract receipt data as JSON with keys:
    store_name, date, items (list of {name, price}), total, tax""",
    img
])
data = json.loads(response.text)
print(f"Store: {data['store_name']}, Total: ${data['total']}")

Audio Transcription

Python — transcribe audio file
import google.generativeai as genai

# Upload audio file (mp3, wav, ogg, etc.)
audio_file = genai.upload_file("interview.mp3", mime_type="audio/mp3")
print(f"Uploaded: {audio_file.uri}")

model = genai.GenerativeModel("gemini-1.5-pro")

# Transcribe
response = model.generate_content([
    audio_file,
    "Transcribe this audio verbatim. Include speaker labels if there are multiple speakers."
])
print(response.text)

# Summarize instead
summary = model.generate_content([
    audio_file,
    "Summarize the key points discussed in this audio in bullet points."
])
print(summary.text)

# Clean up uploaded file
genai.delete_file(audio_file.name)

Video Analysis

Python — analyze video file
import time

# Upload video (supports mp4, mov, avi, etc.)
video_file = genai.upload_file("demo.mp4", mime_type="video/mp4")

# Wait for processing
while video_file.state.name == "PROCESSING":
    print("Processing...")
    time.sleep(5)
    video_file = genai.get_file(video_file.name)

if video_file.state.name == "FAILED":
    raise ValueError("Video processing failed")

model = genai.GenerativeModel("gemini-1.5-pro")

response = model.generate_content([
    video_file,
    "Describe what happens in this video. Include timestamps for key moments."
])
print(response.text)

genai.delete_file(video_file.name)
💡 Context Window

Gemini 1.5 Pro has a 2M token context window - enough for ~2 hours of video, ~22 hours of audio, or 2,000 pages of text in a single call.
