Few-Shot Prompting — Teach by Example

Use few-shot examples to reliably control LLM output format, style, and reasoning patterns.

What Is Few-Shot Prompting?

Instead of describing what you want, you show 2-5 examples of input/output pairs. The LLM learns the pattern and applies it to your real input. It's far more reliable than natural language instructions alone.

Python - zero-shot vs few-shot comparison
# Zero-shot: describe what you want (less reliable)
zero_shot = ask(
    "Convert informal English to formal English.",
    "gonna be late cuz of traffic lol"
)

# Few-shot: show examples (much more reliable)
few_shot_prompt = """Convert informal text to formal business English.

Informal: hey can u send that report asap its urgent
Formal: Could you please send that report at your earliest convenience? This is a time-sensitive matter.

Informal: gonna be late cuz of traffic lol
Formal: I will be arriving late due to traffic congestion. I apologize for any inconvenience.

Informal: thx 4 the help yesterday really saved me
Formal: Thank you for your assistance yesterday. Your support was greatly appreciated.

Informal: {text}
Formal:"""

result = ask("", few_shot_prompt.format(text="gonna be late cuz of traffic lol"))
print(result)  # Consistently formal output

Building a Reusable Few-Shot Template

Python - dynamic few-shot examples
from typing import TypedDict

class Example(TypedDict):
    input: str
    output: str

def few_shot(
    task_desc: str,
    examples: list[Example],
    real_input: str
) -> str:
    shots = "\n\n".join(
        f"Input: {ex['input']}\nOutput: {ex['output']}"
        for ex in examples
    )
    prompt = f"{task_desc}\n\n{shots}\n\nInput: {real_input}\nOutput:"
    return ask("", prompt)

# Sentiment classification
result = few_shot(
    task_desc="Classify the sentiment. Reply with ONLY: positive, negative, or neutral.",
    examples=[
        {"input": "I love this product!", "output": "positive"},
        {"input": "Terrible experience, avoid.", "output": "negative"},
        {"input": "Arrived on time.", "output": "neutral"},
        {"input": "Best purchase ever!", "output": "positive"},
    ],
    real_input="Shipping was slow but product quality is okay."
)
print(result)  # neutral

How Many Examples?

  • 1-2 examples — for simple, consistent format (JSON extraction, classification)
  • 3-5 examples — for complex style transformations, nuanced tasks
  • 5-10 examples — for tasks with many edge cases or ambiguous categories
  • 10+ examples — consider fine-tuning instead; few-shot has diminishing returns beyond 10
💡 Example diversity matters

Include examples that cover edge cases, not just the easy cases. If classifying support tickets, include borderline examples that could go either way — those are what the LLM will get wrong without guidance.
