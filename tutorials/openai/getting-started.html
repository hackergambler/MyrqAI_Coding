Getting Started with OpenAI API

Learn how to create an OpenAI account, generate API keys, set up your development environment, and make your very first API call. This is the foundation for everything else you'll learn.

📑 In This Lesson
  • 1. Create an OpenAI Account
  • 2. Generate Your API Key
  • 3. Set Up Your Environment
  • 4. Make Your First API Call
  • 5. Understanding the Response
  • 6. Next Steps

1. Create an OpenAI Account

Before you can use the OpenAI API, you need to create an account. Here's how:

  1. Go to platform.openai.com
  2. Click "Sign Up" in the top right corner
  3. You can sign up with your email, Google account, Microsoft account, or Apple ID
  4. Verify your email address
  5. Complete phone verification (required for API access)
💡 Pro Tip

New accounts typically receive free credits to get started. Check your usage page to see your current balance.

2. Generate Your API Key

Your API key is like a password that identifies you to OpenAI's servers. Here's how to get one:

  1. Log in to platform.openai.com
  2. Click on your profile icon in the top right
  3. Select "View API Keys" or go directly to platform.openai.com/api-keys
  4. Click "Create new secret key"
  5. Give it a descriptive name (e.g., "My First Project")
  6. Copy the key immediately - you won't be able to see it again!
⚠️ Important Security Note

Never share your API key publicly or commit it to version control. Treat it like a password. If you accidentally expose it, delete it immediately and create a new one.

3. Set Up Your Environment

We'll use Python for this tutorial. First, make sure you have Python 3.8+ installed, then install the OpenAI library:

# Install the OpenAI Python library
pip install openai

Next, set your API key as an environment variable. This is the safest way to use your key:

# On macOS/Linux (add to ~/.bashrc or ~/.zshrc)
export OPENAI_API_KEY="your-api-key-here"

# On Windows (Command Prompt)
set OPENAI_API_KEY=your-api-key-here

# On Windows (PowerShell)
$env:OPENAI_API_KEY="your-api-key-here"

4. Make Your First API Call

Now for the exciting part! Let's make your first API call to GPT-4:

from openai import OpenAI

# Initialize the client (automatically uses OPENAI_API_KEY env variable)
client = OpenAI()

# Make a chat completion request
response = client.chat.completions.create(
    model="gpt-4o-mini",  # Fast and cost-effective model
    messages=[
        {
            "role": "system",
            "content": "You are a helpful assistant."
        },
        {
            "role": "user",
            "content": "Hello! What can you help me with?"
        }
    ]
)

# Print the assistant's response
print(response.choices[0].message.content)

Save this code as first_call.py and run it:

python first_call.py

You should see a friendly response from the AI! 🎉

5. Understanding the Response

The API returns a response object with several useful fields:

# The response object contains:
response.id                              # Unique identifier for this request
response.model                           # The model that was used
response.choices[0].message.content     # The actual response text
response.choices[0].message.role        # Always "assistant"
response.usage.prompt_tokens             # Tokens in your input
response.usage.completion_tokens         # Tokens in the response
response.usage.total_tokens              # Total tokens used
💡 What are Tokens?

Tokens are pieces of words. On average, 1 token ≈ 4 characters or ¾ of a word. The API charges based on tokens used. We'll cover this in detail in the next lesson.
