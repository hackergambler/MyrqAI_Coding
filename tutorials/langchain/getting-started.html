Getting Started with LangChain

Install LangChain, wire up LLM providers, build chains with the LCEL pipe operator, and understand prompts, parsers, and memory.

1. Installation & Setup

bash
# Core LangChain
pip install langchain langchain-core langchain-community

# Provider packages (install the ones you need)
pip install langchain-openai       # OpenAI + Azure
pip install langchain-anthropic    # Claude
pip install langchain-google-genai # Gemini
pip install langchain-ollama       # Local models

# Utilities
pip install python-dotenv          # .env file support
pip install faiss-cpu              # vector store for RAG
Python — .env setup
# .env file
OPENAI_API_KEY=sk-...
ANTHROPIC_API_KEY=sk-ant-...
LANGCHAIN_API_KEY=ls__...       # LangSmith tracing (optional)
LANGCHAIN_TRACING_V2=true

# Load in Python
from dotenv import load_dotenv
load_dotenv()
import os
api_key = os.getenv("OPENAI_API_KEY")

2. Connecting LLM Providers

LangChain abstracts all LLM providers behind a unified interface. Swap providers by changing one import.

Python — multiple providers
from langchain_openai import ChatOpenAI
from langchain_anthropic import ChatAnthropic
from langchain_google_genai import ChatGoogleGenerativeAI

# OpenAI
gpt = ChatOpenAI(model="gpt-4o", temperature=0.7)

# Anthropic Claude
claude = ChatAnthropic(model="claude-opus-4-5", temperature=0.5)

# Google Gemini
gemini = ChatGoogleGenerativeAI(model="gemini-1.5-pro")

# All share the same interface
from langchain_core.messages import HumanMessage

msg = HumanMessage(content="What is 2+2?")
print(gpt.invoke([msg]).content)
print(claude.invoke([msg]).content)
print(gemini.invoke([msg]).content)

3. Prompt Templates

Python — ChatPromptTemplate
from langchain_core.prompts import ChatPromptTemplate

# Basic template
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a {role}. Respond in {language}."),
    ("human", "{question}")
])

# Format it
formatted = prompt.format_messages(
    role="Python expert",
    language="English",
    question="What are list comprehensions?"
)

# Use it
llm = ChatOpenAI(model="gpt-4o-mini")
response = llm.invoke(formatted)
print(response.content)

# FewShotChatMessagePromptTemplate
from langchain_core.prompts import FewShotChatMessagePromptTemplate

examples = [
    {"input": "Great product!", "output": "positive"},
    {"input": "Terrible service.", "output": "negative"},
]
example_prompt = ChatPromptTemplate.from_messages([
    ("human", "{input}"), ("ai", "{output}")
])
few_shot = FewShotChatMessagePromptTemplate(examples=examples, example_prompt=example_prompt)
final_prompt = ChatPromptTemplate.from_messages([
    ("system", "Classify sentiment."), few_shot, ("human", "{input}")
])

4. LCEL — The Pipe Operator

LangChain Expression Language (LCEL) uses the | pipe operator to chain components. Each component's output becomes the next component's input.

Python — LCEL chains
from langchain_openai import ChatOpenAI
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.output_parsers import StrOutputParser

llm = ChatOpenAI(model="gpt-4o-mini")
parser = StrOutputParser()

# Simple chain: prompt | llm | parser
chain = (
    ChatPromptTemplate.from_template("Translate to French: {text}")
    | llm
    | parser
)
result = chain.invoke({"text": "Hello, world!"})
print(result)  # Bonjour, monde!

# Batch invoke
results = chain.batch([
    {"text": "Good morning"},
    {"text": "Thank you"},
    {"text": "Goodbye"},
])
print(results)

# Streaming
for chunk in chain.stream({"text": "The quick brown fox"}):
    print(chunk, end="", flush=True)
Python — branching chains with RunnableBranch
from langchain_core.runnables import RunnableBranch, RunnablePassthrough

classify_chain = (
    ChatPromptTemplate.from_template("Is this about: python, javascript, or other? Answer one word.\n{code}")
    | llm | parser
)

python_chain = ChatPromptTemplate.from_template("Explain this Python: {code}") | llm | parser
js_chain = ChatPromptTemplate.from_template("Explain this JavaScript: {code}") | llm | parser
other_chain = ChatPromptTemplate.from_template("Explain this code: {code}") | llm | parser

router = RunnableBranch(
    (lambda x: "python" in x["topic"].lower(), python_chain),
    (lambda x: "javascript" in x["topic"].lower(), js_chain),
    other_chain
)

full_chain = {"topic": classify_chain, "code": RunnablePassthrough()} | router

5. Output Parsers

Python — JSON and Pydantic parsers
from langchain_core.output_parsers import JsonOutputParser
from langchain_core.pydantic_v1 import BaseModel, Field
from typing import List

# Pydantic schema
class Recipe(BaseModel):
    name: str = Field(description="Recipe name")
    ingredients: List[str] = Field(description="List of ingredients")
    steps: List[str] = Field(description="Cooking steps")
    time_minutes: int = Field(description="Total time in minutes")

parser = JsonOutputParser(pydantic_object=Recipe)

prompt = ChatPromptTemplate.from_messages([
    ("system", "Generate a recipe.\n{format_instructions}"),
    ("human", "Make a recipe for: {dish}")
]).partial(format_instructions=parser.get_format_instructions())

chain = prompt | llm | parser
recipe = chain.invoke({"dish": "pasta carbonara"})
print(recipe["name"], "-", recipe["time_minutes"], "min")

6. Conversation Memory

Python — conversation with history
from langchain_community.chat_message_histories import ChatMessageHistory
from langchain_core.chat_history import BaseChatMessageHistory
from langchain_core.runnables.history import RunnableWithMessageHistory

store = {}

def get_session_history(session_id: str) -> BaseChatMessageHistory:
    if session_id not in store:
        store[session_id] = ChatMessageHistory()
    return store[session_id]

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful coding assistant."),
    ("placeholder", "{history}"),
    ("human", "{input}")
])

chain = prompt | llm | parser
with_history = RunnableWithMessageHistory(
    chain, get_session_history,
    input_messages_key="input",
    history_messages_key="history"
)

config = {"configurable": {"session_id": "user_123"}}
print(with_history.invoke({"input": "What is a decorator?"}, config=config))
print(with_history.invoke({"input": "Show me an example."}, config=config))
