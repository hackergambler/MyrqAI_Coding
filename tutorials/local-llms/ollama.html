Run LLMs Locally with Ollama

Ollama makes running LLMs on your own machine as easy as Docker. Install it, pull models like Llama 3 or Mistral, and call them from Python — zero API costs, full privacy.

Install Ollama

bash - install Ollama
# macOS / Linux
curl -fsSL https://ollama.com/install.sh | sh

# Windows: download installer from ollama.com

# Verify installation
ollama --version

# Pull and run models (downloads automatically)
ollama run llama3.2          # 3B params, ~2GB - fastest
ollama run llama3.1:8b       # 8B params, ~5GB - balanced
ollama run mistral           # 7B params, ~4GB - great for code
ollama run codellama:13b     # 13B params, ~8GB - code specialist
ollama run phi3:mini         # 3.8B params, ~2GB - Microsoft model
ollama run qwen2.5:7b        # 7B params, ~5GB - excellent multilingual

# List downloaded models
ollama list

# Check running models
ollama ps

Python Integration

Python - call Ollama via REST API
import requests
import json

# Ollama runs a local server on port 11434
BASE_URL = "http://localhost:11434"

def generate(prompt: str, model: str = "llama3.2") -> str:
    resp = requests.post(f"{BASE_URL}/api/generate", json={
        "model": model,
        "prompt": prompt,
        "stream": False
    })
    return resp.json()["response"]

def chat(messages: list, model: str = "llama3.2") -> str:
    resp = requests.post(f"{BASE_URL}/api/chat", json={
        "model": model,
        "messages": messages,
        "stream": False
    })
    return resp.json()["message"]["content"]

print(generate("Explain recursion in one paragraph."))

print(chat([
    {"role": "system", "content": "You are a Python expert."},
    {"role": "user", "content": "Write a merge sort in Python."}
]))
Python — OpenAI-compatible Ollama client
from openai import OpenAI

# Ollama is OpenAI-compatible — just change the base_url!
client = OpenAI(
    base_url="http://localhost:11434/v1",
    api_key="ollama"  # any string works
)

response = client.chat.completions.create(
    model="llama3.2",
    messages=[
        {"role": "system", "content": "You are a coding assistant."},
        {"role": "user", "content": "Write a binary search in Python."}
    ]
)
print(response.choices[0].message.content)

# Streaming works too
stream = client.chat.completions.create(
    model="mistral",
    messages=[{"role": "user", "content": "Tell me a story."}],
    stream=True
)
for chunk in stream:
    print(chunk.choices[0].delta.content or "", end="", flush=True)

Choosing the Right Model

  • llama3.2 (3B) — Fast, runs on any machine with 8GB RAM. Great for quick tasks.
  • llama3.1:8b — Best balance of speed and quality. Needs 16GB RAM.
  • mistral:7b — Excellent at coding and instruction following.
  • codellama:13b — Specialized for code generation and completion.
  • qwen2.5:7b — Top multilingual model. Supports 29 languages.
  • phi3:mini — Microsoft's small but capable model. Low memory usage.
Python — benchmark models for your task
import time

models = ["llama3.2", "mistral", "phi3:mini"]
test_prompt = "Write a Python function to calculate Fibonacci numbers using memoization."

for model in models:
    start = time.time()
    result = generate(test_prompt, model=model)
    elapsed = time.time() - start
    tokens = len(result.split())
    print(f"\n=== {model} ({elapsed:.1f}s, ~{tokens} words) ===")
    print(result[:200], "...")
