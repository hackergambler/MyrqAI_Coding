Fine-tuning Local LLMs with LoRA

Fine-tune open-source LLMs with LoRA/QLoRA using the Unsloth library. Train on custom data in minutes.

Why Fine-tune?

Fine-tuning adapts a base model to your specific domain, format, or style. With LoRA (Low-Rank Adaptation) you update only 0.1-1% of parameters — fast, cheap, and fits on a single GPU.

  • LoRA — train small adapter matrices, freeze the base model
  • QLoRA — LoRA on a 4-bit quantized base model — 4x less VRAM
  • Unsloth — 2x faster training, 60% less VRAM than standard approaches
bash - install Unsloth
pip install unsloth
# or for specific CUDA versions:
pip install "unsloth[cu121-torch230] @ git+https://github.com/unslothai/unsloth.git"
Python - QLoRA fine-tuning with Unsloth
from unsloth import FastLanguageModel
from datasets import Dataset
from trl import SFTTrainer
from transformers import TrainingArguments

# Load base model with 4-bit quantization
model, tokenizer = FastLanguageModel.from_pretrained(
    model_name="unsloth/llama-3.2-3b-instruct",
    max_seq_length=2048,
    load_in_4bit=True,
)

# Add LoRA adapters
model = FastLanguageModel.get_peft_model(
    model,
    r=16,                  # LoRA rank — higher = more params, better quality
    lora_alpha=16,
    target_modules=["q_proj","k_proj","v_proj","o_proj"],
    use_gradient_checkpointing="unsloth",
)

# Your training data
train_data = Dataset.from_list([
    {"text": "<|user|>\nExplain Python decorators\n<|assistant|>\nDecorators are functions that wrap other functions..."},
    {"text": "<|user|>\nWhat is a generator?\n<|assistant|>\nGenerators use yield to produce values lazily..."},
])

# Train
trainer = SFTTrainer(
    model=model, tokenizer=tokenizer,
    train_dataset=train_data,
    dataset_text_field="text",
    max_seq_length=2048,
    args=TrainingArguments(
        output_dir="./finetuned",
        num_train_epochs=3,
        per_device_train_batch_size=2,
        learning_rate=2e-4,
        logging_steps=10,
    )
)
trainer.train()

# Save and test
model.save_pretrained_gguf("./finetuned_gguf", tokenizer)  # export for Ollama!
