HuggingFace Transformers — Local Inference

Run HuggingFace models locally with transformers, pipeline API, and optimized inference with bitsandbytes.

Run Any HuggingFace Model Locally

The transformers library lets you run thousands of open-source models locally with just a few lines of code.

bash - install
pip install transformers torch accelerate
pip install bitsandbytes  # for 4/8-bit quantization
Python - pipeline API (simplest)
from transformers import pipeline

# Text generation with Llama 3.2
gen = pipeline(
    "text-generation",
    model="meta-llama/Llama-3.2-3B-Instruct",
    device_map="auto",      # auto-assign GPU/CPU
    torch_dtype="auto"
)

result = gen(
    [{"role":"user","content":"Explain recursion briefly."}],
    max_new_tokens=256,
    temperature=0.7
)
print(result[0]["generated_text"][-1]["content"])

4-bit Quantization with bitsandbytes

Python - load in 4-bit to save GPU memory
from transformers import AutoTokenizer, AutoModelForCausalLM, BitsAndBytesConfig
import torch

quantization_config = BitsAndBytesConfig(
    load_in_4bit=True,
    bnb_4bit_compute_dtype=torch.float16,
    bnb_4bit_use_double_quant=True
)

model_id = "mistralai/Mistral-7B-Instruct-v0.3"
tokenizer = AutoTokenizer.from_pretrained(model_id)
model = AutoModelForCausalLM.from_pretrained(
    model_id,
    quantization_config=quantization_config,
    device_map="auto"
)

# Generate
messages = [{"role":"user","content":"Write a Python decorator example."}]
inputs = tokenizer.apply_chat_template(messages, return_tensors="pt").to(model.device)
outputs = model.generate(inputs, max_new_tokens=256)
response = tokenizer.decode(outputs[0][inputs.shape[1]:], skip_special_tokens=True)
print(response)

Streaming with HuggingFace

Python - stream tokens as they generate
from transformers import TextIteratorStreamer
from threading import Thread

streamer = TextIteratorStreamer(tokenizer, skip_prompt=True, skip_special_tokens=True)

inputs = tokenizer.apply_chat_template(
    [{"role":"user","content":"Tell me a story."}], return_tensors="pt"
).to(model.device)

# Run generation in background thread
thread = Thread(target=model.generate, kwargs={
    "input_ids": inputs, "streamer": streamer, "max_new_tokens": 256
})
thread.start()

# Stream tokens as they arrive
for token in streamer:
    print(token, end="", flush=True)
thread.join()
💡 Memory guide

A 7B model needs: FP16 = 14GB VRAM, INT8 = 7GB, INT4 = 4GB. If you have 8GB VRAM, use 4-bit quantization for 7B models or run 3B models in FP16.
