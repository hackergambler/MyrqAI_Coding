LangChain with Local LLMs

Run completely local AI pipelines: LangChain + Ollama for text generation, and Nomic/mxbai embeddings for fully private RAG systems.

LangChain + Ollama Setup

bash + Python — install and configure
# Install LangChain Ollama integration
pip install langchain-ollama

# Pull embedding model too
ollama pull nomic-embed-text   # fast local embeddings
ollama pull mxbai-embed-large  # higher quality embeddings
Python - local LLM chain
from langchain_ollama import ChatOllama, OllamaEmbeddings
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.output_parsers import StrOutputParser

# Local LLM — no API key needed
llm = ChatOllama(model="llama3.1:8b", temperature=0.7)

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a concise technical writer. Answer in 2-3 sentences max."),
    ("human", "{question}")
])

chain = prompt | llm | StrOutputParser()

print(chain.invoke({"question": "What is a Python generator?"}))

# Streaming
for chunk in chain.stream({"question": "Explain async/await"}):
    print(chunk, end="", flush=True)

Fully Local RAG Pipeline

Python - 100% local RAG with Ollama + FAISS
from langchain_ollama import ChatOllama, OllamaEmbeddings
from langchain_community.vectorstores import FAISS
from langchain_core.documents import Document
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.output_parsers import StrOutputParser
from langchain_core.runnables import RunnablePassthrough

# Local embeddings (completely private)
embeddings = OllamaEmbeddings(model="nomic-embed-text")

# Your private documents
docs = [
    Document(page_content="Our Q1 revenue was $4.2M, up 23% YoY.", metadata={"source": "q1_report"}),
    Document(page_content="The product roadmap includes: mobile app (Q2), API v2 (Q3), enterprise (Q4)."),
    Document(page_content="Employee headcount: 47 full-time, 12 contractors. Growing 15% QoQ."),
]

# Build local vector store
vectorstore = FAISS.from_documents(docs, embeddings)
retriever = vectorstore.as_retriever(search_kwargs={"k": 2})

# Local LLM for generation
llm = ChatOllama(model="llama3.1:8b")

prompt = ChatPromptTemplate.from_template("""Answer using only this context:
{context}
Question: {question}""")

rag_chain = (
    {"context": retriever, "question": RunnablePassthrough()}
    | prompt | llm | StrOutputParser()
)

# Your data never leaves your machine!
print(rag_chain.invoke("What was Q1 revenue growth?"))
print(rag_chain.invoke("When is the mobile app launching?"))
💡 Privacy Guarantee

With Ollama + FAISS + local embeddings, your data never leaves your machine. Perfect for healthcare records, legal documents, and confidential business data.
